Station météo LoRa autonome

18 Avril 2026 Par antoine8205 8 min de lecture

Le projet consiste à déployer une sonde météo capable de transmettre des données (température, humidité, pression) sur une distance de 10km sans utiliser de Wi-Fi ou de 4G, grâce à la technologie LoRaWAN.

Le Matériel Utilisé

  • Micro-contrôleur ESP32 Heltec LoRa V2
  • Capteur BME280 (Bosch)
  • Panneau solaire 5V avec gestionnaire de charge TP4056
  • Batterie Li-Po 18650

Extrait du Code (Arduino)

#include <LoRa.h>
void setup() {
  Serial.begin(115200);
  if (!LoRa.begin(868E6)) {
    Serial.println("Erreur LoRa");
  }
}

Déploiement sur le terrain

L'installation a été faite sur un mât à 15 mètres d'altitude pour maximiser la portée du signal. Les premiers tests montrent une réception stable à 8,5km en zone péri-urbaine.
